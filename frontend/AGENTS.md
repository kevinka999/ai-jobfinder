# Frontend Agent Instructions

## Design System

Read `docs/DESIGN_SYSTEM.md` before changing frontend UI, component styling, or layout behavior.

Use Tailwind utilities with the tokens from `src/design/tokens.css`. Keep global CSS limited to Tailwind setup, tokens, base element defaults, and rare browser-level behavior.

## App Shape

This is a React and TypeScript Vite app. Keep UI work focused on fast workflow screens for a single-user job application tool.
